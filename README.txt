Description:  
This web app allows users to input a range of rows and columns to dynamically generate a multiplication table. 
It includes real-time validation using the jQuery Validation Plugin, which ensures all inputs are numeric and within the range of -50 to 50. 
Invalid input triggers styled error messages and prevents form submission. 
On valid input, a styled table is generated without page reload.
